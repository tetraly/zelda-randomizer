Update History

V0.05 Updates

    This is the initial version of the web randomizer, which makes it more convenient to randomize ROMs. New features over the 0.04 release include:
  • Added flags for controlling randomization of certain overworld items
  • Added a flag for Swapping items with the select button.
  • Added a flag for randomizing the "LEVEL-" text in dungeons.
  • Added a flag to speed up text scrolling speed.
  • Added a flag for avoiding "hard" combat situations.
  • Fixed bug sometimes causing an item not to appear in a staircase room.
  • Fixed bug that caused consecutitve seed numbers to sometimes produce the same game.
